# [alex.gondek.ca](https://alex.gondek.ca/)

My personal homepage, plus various posts and galleries.

**Build Requirements**
- `ruby` and Bundler
- `just` command runner (see [justfile](./justfile))
- `imagemagick` (for `identify` and `mogrify` commands)

**Local Development**
1. `just install`
2. `just serve`

**Deployment**
1. `git commit` changes
2. `git push`
